How to Add a Company

Follow this simple step-by-step guide to contribute to our directory and showcase your software team.

1

GitHub Permission

Start by forking the repository or asking for access. Most contributions begin with creating a feature branch named add-company-[name].

2

Create JSON File

Navigate to data/companies/ and create a new file: yourcompany.json. Ensure the filename matches the company id.

3

JSON Body Structure

{
  "id": "newcorp",
  "name": "NEW CORP",
  "description": "Enter your company mission and vision here.",
  "icon": "N",
  "iconColor": "bg-indigo-500",
  "logo": "data/logos/newcorp.png",
  "tags": ["SOFTWARE ENG.", "CLOUD", "AI"],
  "employees": 150,
  "location": "Global / Remote",
  "website": "https://www.newcorp.com",
  "phone": "+1 555 000 0000",
  "founded": 2024,
  "openPositions": [
    {
      "title": "Software Engineer",
      "department": "Engineering",
      "location": "Remote",
      "type": "Full-time"
    }
  ]
}

Copy this template and fill it with your company details. Make sure you don't miss commas!

4

Commit & Push

Use Git to upload your changes. Open your terminal and run:

git add .
git commit -m "feat: add NewCorp"
git push origin add-company-newcorp
5

Create Pull Request

Go to GitHub and click "Compare & pull request". Describe your changes and wait for the review. Once merged, your company will be live!
